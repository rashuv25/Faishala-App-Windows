# मुद्दा फैसला (Mudda Phaisala)

## Court Judgement Document Creator

A desktop application for creating court judgement documents using pre-defined templates.

---

## Features

- 📄 Create court judgement documents
- 📝 Fill templates with ease
- 💾 Auto-save on every keystroke
- 🔄 Undo/Redo support (Ctrl+Z / Ctrl+Y)
- 📤 Export to DOCX and PDF
- 🔍 Search and filter documents
- 📚 Dictionary for saved names
- 🔒 Secure login system
- ☁️ Silent backup (when online)

---

## Requirements

- Python 3.10 or higher
- Windows 10/11 or Linux
- Internet connection (for monthly login only)
